# HPCtoolkit
HPCを使用するときに、プログレスバーを表示するライブラリ

# 説明
exampleフォルダのようにHPCtoolkit使用すると、進捗がテキストファイルに記入されます。

tqdmを使用するのが一般的ですが、なぜかリアルタイムで出力されなかったり、出力画面を全て見ることができません。

（明示的にフラッシュすればリアルタイムで出力できるかも　sys.stdout.flush()　）
